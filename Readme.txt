 Model 디렉토리에는, 메터리얼, 텍스처, 메쉬 등의 3D모델 자료를 각 모델 단위로 디렉토리를 내부에 생성하여 보관한다.
예 : Model 내부의 Rock 디렉토리 안에 바위의 모든 메쉬, 메터리얼, 텍스처가 저장되어 있다.

 Sound 디렉토리에는 모든 필요한 음향 에셋을 보관한다. 디렉토리 내부에서 조금 더 내부적으로 BGM,Effect로 분할한다.

 Map 디렉토리에는 디자인된 모든 스테이지 맵들을 보관한다.

 Particle 디렉토리에는 파티클 이펙트가 들어가 있다.

 Blueprint 디렉토리에는 블루프린트들이 들어가 있다. 블루프린트들을 한 디렉토리에 모두 넣는 것은, 레벨디자인의 편의를 도모하기 위함이다.
